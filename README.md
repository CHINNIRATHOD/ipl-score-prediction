# IPL Score Prediction (Machine Learning)

## 📌 Problem Statement
Predict IPL match scores based on match conditions such as runs, wickets, overs, and teams.

## 📊 Dataset
- Source: Kaggle / IPL dataset  
- Size: ~10,000+ records  

## ⚙️ Approach
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering  
- Model Training (Linear Regression, Decision Tree)  

## 📈 Results
- Achieved ~85% prediction accuracy  
- Improved performance using feature engineering  

## 🛠 Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib
